# Food Delivery- by Reactjs

### Features:

---

#### features included in this project

- Add to cart functionality
- Shopping cart functionality
- Product Filtering functionality
- Product page
- Cart and checkout page
- Login and Register page
- Authentication by firebase for login , logout and register a new user
- Register a new user by normal email or email google 
- You can log in using this email sayed@yahoo.com or create a new account
- After logging in, you can view the cart and make a checkout

### Tools Used in This Project:

---

- React.js
- Redux Toolkit
- react-icons
- react-paginate
- react-toastify
- react-slick
- ReactStrap
- Bootstrap
- CSS
- firebase
- slick-carousel


### How to start?

---

##### To start the project along with me just download the start-up file and run available script

```javascript
// first install all necessary dependencies

npm i

// next run

npm start

```
